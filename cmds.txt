# TIPS FOR TEXT EDITORS

# nano
# - Ctrl+O to save
# - Ctrl+X to exit
# - Ctrl+K to cut a line, Ctrl+U to paste

# micro
# - Ctrl+S to save
# - Ctrl+Q to quit
# - Ctrl+E to open command prompt (for plugins etc.)
# - Supports mouse, syntax highlighting, multi-line select

# vim
# - Type `i` to insert
# - Esc + :wq to save & quit
# - Esc + :q! to quit without saving
