Errors
- Compile-time errors //errors found by the compiler
	- Syntax Errors //Errors when messing up C++ rules (Ex. Missing ;,),})
	- Type Errors //Errors when types mismatched (Ex. Forgetting to declare variable, storing value in wrong type)
- Link-time error //errors found by the linker when trying to combine object files to be executable
- Run-time errors //Found by checks in running program (Happen after ./a.out)
	- Division Error (Ex. Division by zero)
	- Trying to open a file that doesn't exist
	- Floating point exception
- Logic Errors //Found by programmer looking for causes of incorrect results
